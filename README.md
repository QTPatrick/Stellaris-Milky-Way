# Stellaris-Milky-Way
This mod aims to create a semi-static Milky Way map as a galaxy creation setting.
It will probably only work for single-player Earth starts.
AI empires will consist of primarily preset empires, positioned according to their lore (e.g. Blorg are close to Earth).
It will force all real stars included in the random name files to spawn relative to their real locations (with limited precision or dozens of stars will occupy the same pixel).
It will evenly spread out ruined megastructures and unique systems, but in a way that accelerates player access to the rest of the galaxy.
AI will receive additional starting bonuses to compensate for player advantages and encourage the sense that Earth is joining an already dynamic galactic community.
Pre-FTL societies will be few, but will connect to lore of surrounding precursors, fallen empires and unique systems.
